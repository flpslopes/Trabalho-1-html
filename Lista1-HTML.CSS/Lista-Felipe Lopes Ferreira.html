exercício 3: Lista 1 - Questão 3

Diferença entre Linguagem Server Side e Client Side

Server Side é executado no servidor, processando dados e gerando conteúdo dinâmico.

Client Side roda no navegador, manipulando a interface do usuário.

Exemplos:

  • Server Side: PHP, Python , Ruby, Java.
  • Client Side: JavaScript, HTML, CSS.

O que é Semântica no HTML5?

Semântica é o estudo do significado e da interpretação das palavras. Aplicar esse conceito ao HTML significa que buscamos um melhor entendimento do conteúdo por parte dos computadores e usuários, portanto o HTML Semântico é um método de codificação que prioriza o significado e o contexto das tags HTML
